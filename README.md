# 睡眠健康数据分析仪表板 📊😴

基于Streamlit构建的交互式多页面睡眠健康数据分析仪表板，整合24张可视化图表，采用浅色主题设计，适合数据汇报和展示。

## 项目简介

本项目对睡眠健康与生活方式数据集进行深入分析，通过数据可视化和统计分析，揭示影响睡眠质量的关键因素，包括：

- 🏃 生活方式因素（运动、职业压力）
- 💔 健康风险指标（BMI、血压、心率）
- 👥 人群差异特征（性别、年龄）
- 🔬 睡眠障碍深度分析

## 功能特性

### 📈 多页面导航
- **首页**: 关键指标卡片、核心图表、数据洞察
- **生活方式分析**: 运动与睡眠、职业压力分析
- **健康风险评估**: BMI、血压、心率风险评估
- **人群差异洞察**: 性别、年龄趋势分析
- **深度探索**: 睡眠障碍分析、原始数据浏览

### 🎨 界面特色
- ✅ 浅色主题设计，适合汇报展示
- ✅ 响应式布局，支持多种屏幕尺寸
- ✅ 交互式筛选器（性别、职业、年龄段）
- ✅ 数据下载功能（CSV/Excel格式）
- ✅ 自动洞察生成

### 📊 24张可视化图表
- 相关性热力图
- 特征重要性分析
- 运动与睡眠回归分析
- 职业压力箱线图
- BMI与睡眠障碍分析
- 多维度雷达图
- 更多深度分析图表...

## 技术栈

- **框架**: Streamlit
- **数据处理**: Pandas, NumPy
- **数据可视化**: Matplotlib, Seaborn
- **机器学习**: Scikit-learn (LabelEncoder, RandomForestRegressor)
- **主题**: 自定义浅色主题

## 使用指南

### 1. 首页查看关键指标
- 查看平均睡眠质量、睡眠障碍比例等关键指标
- 阅读自动生成的数据洞察
- 浏览核心图表（相关性热力图、特征重要性）

### 2. 使用侧边栏筛选
- 选择性别、职业、年龄范围
- 筛选后的数据会实时更新相关指标

### 3. 探索专题页面
- 点击侧边栏导航到不同的专题分析页面
- 每个页面都包含详细的图表说明和数据洞察

### 4. 下载数据
- 在各个页面底部可以下载筛选后的数据
- 支持CSV和Excel格式
